== Description ==

Allingroups lets you automatically publish a message, as if you were writing each one individually, to all of the Facebook groups that you select.

In a few clicks, you can start a campaign that you can follow in real time on your [dashboard](http://connect.allingroups.com/ "Allingroups dashboard").

You can also save your group playlists so that you can use them again in new campaigns. Additionally, you can join groups matching your search criteria automatically.

IMPORTANT: This plugin requires a valid Facebook account, the plugin will ask it and use it to create an account remotely on http://connect.allingroups.com/ , so the plugin will send your informations through the network in https requests to allingroups API.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/allingroups` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Use the Settings > Allingroups screen to configure the plugin.
1. Connect with a Facebook account.
1. Save.
1. Set the publish mode (auto or selector).
1. Set the format to publish (full or trailer).
1. Set the groups selection (or create one).
1. Save again.
